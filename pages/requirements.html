Study Abroad Requirements

Explore admission requirements, English language tests and documents needed for international universities.

Country Wise Requirements

Australia

  • IELTS / PTE / TOEFL
  • Academic Transcript
  • Passport
  • Statement of Purpose
  • Financial Documents

USA

  • IELTS / TOEFL / Duolingo
  • Academic Transcript
  • SAT / GRE (sometimes)
  • Recommendation Letter
  • Personal Essay

United Kingdom

  • IELTS / PTE
  • Academic Transcript
  • Passport
  • Statement of Purpose
  • CV / Resume

Germany

  • IELTS / TOEFL
  • German Language Test (sometimes)
  • Academic Transcript
  • APS Certificate
  • CV / Resume

Japan

  • IELTS / TOEFL
  • JLPT for Japanese Programmes
  • Academic Transcript
  • Passport
  • Recommendation Letter

Canada

  • IELTS / PTE / TOEFL
  • Academic Transcript
  • Passport
  • Statement of Purpose
  • Financial Proof

New Zealand

  • IELTS / PTE / TOEFL
  • Academic Transcript
  • Passport
  • Statement of Purpose
  • Medical Documents

Important Note

Requirements can change depending on university, faculty and degree level.

Always visit official university websites for latest updates before applying.
